TokenCost Monitor

实时监控你的 AI API 消耗和花费

TokenCost Monitor 是一款桌面悬浮窗工具，可实时显示你的 AI API 余额、今日消费，让你随时掌握使用情况，不用在进入网页进行查看，避免超支。

↓ 开源下载 TokenCost Monitor

快速上手

系统要求

  • Windows / macOS / Linux
  • Node.js ≥ 18
  • Python ≥ 3.8（需要 flask 和 flask-cors）

安装步骤

  1. 下载并解压软件
  2. 安装依赖（Node + Python）
  3. 配置 API Key（DeepSeek / OpenAI / Anthropic）
  4. 启动软件即可开始监控
Tip: 软件会自动记录所有通过本地代理的 API 请求，生成详细消费报表。

功能亮点

1. 实时余额与费用监控

  • 查看账户余额
  • 今日 / 本周 / 本月消费金额
  • 支持多模型独立余额预警

2. 自动记录 API 消耗

将你的 LLM 客户端（如 Cursor、Claude Code）API Base 设置为本地代理：

http://127.0.0.1:5099/v1

所有请求自动统计 Token 使用和费用

3. 桌面窗口功能

  • 固定窗口：点击图钉按钮，窗口保持在桌面上，方便随时查看
  • 开机自启：在设置面板一键开启

4. 多账户管理

  • 支持最多 3 组 API Key
  • 每组 Key 可单独设置余额预警

使用提示

  • 余额显示 "--"？检查 API Key 是否正确，网络是否能访问 API
  • 代理不工作？确认软件启动成功，并将客户端 API Base 设置为本地地址
  • 取消开机自启：在设置面板关闭，或删除启动项快捷方式

总结

TokenCost Monitor 让 AI 开发者和使用者轻松掌握 API 消费情况，不再担心超支或浪费，每次调用都清清楚楚。
