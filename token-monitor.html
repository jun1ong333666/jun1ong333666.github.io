TokenCost Monitor

实时监控你的 AI API 消耗和花费

TokenCost Monitor 是一款桌面悬浮窗工具，可实时显示你的 AI API 余额、今日消费，让你随时掌握使用情况，不用在进入网页进行查看，避免超支。

↓ 开源下载 TokenCost Monitor
🚀

快速上手

1

下载运行

下载并解压后，直接启动：

token-monitor.exe

程序会自动启动本地代理服务。

2

配置 API

点击右上角 ⚙️ 设置按钮，填写：

  • API Key
  • API Base URL
  • 余额预警
  • 月度预算

支持最多 3 组 API 配置。

✨

功能亮点

📊

实时消费统计

自动统计：

  • Token 用量
  • 请求次数
  • 今日 / 本周 / 本月消费
  • 模型费用占比
🔔

余额与预算预警

支持：

  • DeepSeek 余额查询
  • 低余额提醒
  • 月度预算监控

帮助控制 API 成本。

📈

可视化数据分析

内置趋势图与模型统计：

  • Token 趋势图
  • 模型占比图
  • 多时间维度查看

所有数据实时更新。

📌

使用提示

  • 建议长期后台运行，方便持续统计数据
  • 余额自动查询目前仅支持 DeepSeek
  • 配置与历史数据保存在：
    config.json
    data/usage.db

备份即可迁移全部数据。

✅

总结

Token Monitor 是一款轻量、开源的 AI API 用量监控工具。 无需修改业务代码，即可实时追踪 Token 消耗与 API 费用，帮助你更直观地管理 AI 成本。
